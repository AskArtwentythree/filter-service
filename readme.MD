# Filter Service

Filter Service for an event-driven system built using RabbitMQ and Python.

## Requirements

- Docker
- Python 3.x

## Setup

1. Clone the repository:

2. Build the Docker containers:

```bash
docker-compose build
```

3. Start the services:

```bash
docker-compose up
```

4. To stop the running services:

```bash
docker-compose down
```
